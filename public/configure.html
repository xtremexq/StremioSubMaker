üìñ How to Use SubMaker

√ó

üöÄ Instructions:

  1. Add desired API keys to "Subtitles Sources" section.
  2. Add a Gemini API Key to AI section.
  3. Select up to 3 source languages.
  4. Select target languages.
  5. Install on Stremio.

üé¨ Stremio will:

Fetch all source and target languages subtitles from sources.
Create "Make [Target Language]" buttons within the subtitles list.

‚ö° When you click a subtitle:

Within the translation list, it'll start translating.
This may take from 1-10 minutes depending on selected AI model and movie/episode length (flash-lite recommended for 1-3 min).

After a few minutes, click again at the same subtitle entry.
It'll load, translated.

*Only 1 source language recommended, BECAUSE:*

It'll then load the subtitles list in the same order within the source language and the "Make [Language]" button, so you can test source subtitles for correct sync and select the same one on the translation list instead of translating unsync'd subtitles.

PS: This addon DOESN'T work without API Keys.

üìÅ Translate SRT Instructions

√ó

üì§ How to Use "Translate SRT":

  1. Enable it on the config page before installing.
  2. Click "Translate SRT" in the subtitles list when streaming.
  3. Right-click the stream or select the three-dots menu and click "Download Subtitles"
  4. A webpage will open on your browser.
  5. Upload the .srt file you'd like to translate. It will use your Gemini API key.
  6. When it's done, download the translated subtitles and drag it to Stremio.
