SubMaker guide

How SubMaker works in Stremio

Set your source-language order first, then use the matching "Make" entries in Stremio to translate from the source subtitle you already checked.

Basic setup

  1. Enable the subtitle sources you want in Subtitles Sources.
  2. Add your Gemini API key in the AI section.
  3. Select source languages. One source language is usually best.
  4. Select target languages for the "Make [Language]" lists.
  5. Install in Stremio and open any video's subtitles list.

What SubMaker adds in Stremio

1. Fetches and shows subtitles in your selected source and target languages from the enabled subtitle providers.
2. Creates "Make [Target Language]" lists inside Stremio's subtitle picker.
3. Each "Make" list keeps the same order as the selected source-language list.

When you use a "Make" subtitle entry

Click a subtitle inside a "Make" list and SubMaker starts translating from the matching source subtitle.
After about 20-30 seconds, reload that same subtitle entry to see the translated version.

Recovery tip

If a translation looks broken, wrong, or out of sync while the source subtitle is fine, triple-click that same subtitle entry to force a fresh re-translation, or change translation settings.

Translation timing

Partial translation arrives in batches so you can start watching quickly. Full translation can still take 30 seconds to 5 minutes, depending on the length of the movie or episode.

Only 1 source language is recommended

Because the "Make" lists follow the same order as the source-language subtitles, you can test the source subtitles for sync first, then choose the matching entry in the translation list instead of translating a source subtitle that is already out of sync.

Sync tip: Check the middle and end of the source subtitles before translating, because sync drift often shows up there first.

🧰 Sub Toolbox Instructions

×

🚀 How to use Sub Toolbox:

  1. In Stremio, open the subtitles list and click "Sub Toolbox".
  2. Right-click the playing stream/video in Stremio and choose "Download Subtitles" to launch the toolbox in your browser.
  3. Pick a tool: Extract & translate embedded subs, Translate subtitle files, Synchronize subtitles, or Automatic subtitles.
  4. Each tool opens in your browser using your saved SubMaker session and AI provider settings.
  5. Follow the on-screen steps, then drag the result back into Stremio if needed.
